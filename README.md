IrisOut Calc adalah aplikasi kalkulator berbasis web yang menggabungkan presisi logika aritmatika dengan estetika desain modern. Dibangun sepenuhnya menggunakan JavaScript, HTML dan Tailwind CSS, proyek ini mendemonstrasikan manipulasi DOM tingkat lanjut tanpa ketergantungan pada framework JavaScript berat. Fokus utama pengembangan adalah menciptakan pengalaman pengguna (User Experience) yang mulus melalui fitur Chain Calculation, yang memungkinkan operasi matematika berantai diproses secara akurat tanpa terputus. Selain itu, antarmuka pengguna mengusung tema Red Gradient dengan transisi animasi "Iris Reveal" yang dinamis, serta tata letak responsif yang beradaptasi sempurna pada perangkat desktop maupun seluler.
